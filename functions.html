Python Functions

Learn how to create reusable blocks of code with Python functions.

Defining Functions

Functions are defined using the def keyword followed by the function name and parentheses.

# Basic function definition
def greet():
    print("Hello, World!")

# Calling the function
greet()

Parameters and Arguments

Functions can accept parameters to make them more flexible and reusable.

# Function with parameters
def greet(name):
    print(f"Hello, {name}!")

# Calling with argument
greet("Alice")

Default Parameters

# Function with default parameter
def greet(name="World"):
    print(f"Hello, {name}!")

greet()        # Uses default
greet("Bob")   # Overrides default

Return Values

Functions can return values using the return statement.

# Function with return value
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # Output: 8

Variable Scope

Understanding local vs. global variables is crucial in function programming.

# Global vs local variables
x = 10  # Global variable

def my_function():
    y = 5  # Local variable
    print(x)  # Can access global
    print(y)

my_function()
print(x)  # Works
# print(y)  # Error - y is local to the function
