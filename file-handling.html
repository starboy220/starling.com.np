File Handling in Python

Learn how to work with files in Python - reading, writing, and managing file operations.

Opening and Closing Files

The basic operations for file handling.

# Opening a file (automatically closes with 'with')
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Equivalent to:
file = open('example.txt', 'r')
try:
    content = file.read()
    print(content)
finally:
    file.close()

Reading Files

Different methods for reading file content.

# Read entire file
with open('example.txt', 'r') as file:
    content = file.read()

# Read line by line
with open('example.txt', 'r') as file:
    for line in file:
        print(line.strip())

# Read all lines into a list
with open('example.txt', 'r') as file:
    lines = file.readlines()

Writing to Files

Creating and modifying files.

# Writing to a file (overwrites existing content)
with open('output.txt', 'w') as file:
    file.write("Hello, World!\n")
    file.write("This is a second line.")

# Appending to a file
with open('output.txt', 'a') as file:
    file.write("\nThis line was appended.")

Working with File Paths

Using the os and pathlib modules.

import os
from pathlib import Path

# Using os module
print(os.path.join('folder', 'subfolder', 'file.txt'))
print(os.path.exists('example.txt'))

# Using pathlib (modern approach)
path = Path('folder') / 'subfolder' / 'file.txt'
print(path.exists())
print(path.suffix)  # .txt
print(path.stem)    # file

Working with CSV and JSON

Handling common file formats.

import csv
import json

# Reading CSV
with open('data.csv', 'r') as file:
    reader = csv.DictReader(file)
    for row in reader:
        print(row['name'], row['email'])

# Writing JSON
data = {'name': 'Alice', 'age': 25, 'city': 'New York'}
with open('data.json', 'w') as file:
    json.dump(data, file, indent=4)
