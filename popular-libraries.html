Popular Python Libraries

Explore the most useful and widely-used Python libraries for various domains.

Web Development

Flask

Lightweight web framework for building APIs and web applications.

from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, World!"

if __name__ == '__main__':
    app.run()

Data Science & Machine Learning

NumPy

Fundamental package for scientific computing with Python.

import numpy as np

arr = np.array([1, 2, 3])
print(arr * 2)  # [2 4 6]

Pandas

Powerful data analysis and manipulation tool.

import pandas as pd

data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)
print(df)

Matplotlib

Comprehensive library for creating static, animated, and interactive visualizations.

import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [4, 5, 1])
plt.show()

Utility Libraries

Requests

Elegant and simple HTTP library for Python.

import requests

response = requests.get('https://api.github.com')
print(response.status_code)  # 200

BeautifulSoup

Library for web scraping and parsing HTML/XML.

from bs4 import BeautifulSoup

soup = BeautifulSoup(html_doc, 'html.parser')
print(soup.title.text)
