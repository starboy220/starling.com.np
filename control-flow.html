Control Flow

Learn how to control the execution flow of your Python programs with conditionals and loops.

If Statements

Conditional statements allow you to execute code only when certain conditions are met.

# Basic if statement
age = 18

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

Loops

Loops allow you to execute a block of code repeatedly.

For Loop

# Iterate through a sequence
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

While Loop

# Execute while condition is true
count = 0
while count < 5:
    print(count)
    count += 1

Break and Continue

Control loop execution with break and continue statements.

# Using break and continue
for num in range(10):
    if num == 3:
        continue  # Skip this iteration
    if num == 8:
        break     # Exit the loop
    print(num)
