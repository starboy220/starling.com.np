Python Basics

Learn the fundamental concepts that form the foundation of Python programming.

Variables

Variables are used to store data in Python. Unlike many other languages, Python doesn't require explicit declaration of variables.

# Variable assignment
name = "Alice"
age = 25
is_student = True

Data Types

Python has several built-in data types:

  • Numbers: Integers (int), Floating point (float)
  • Strings: Text data (str)
  • Booleans: True or False (bool)
# Different data types
number = 42            # int
pi = 3.14159           # float
message = "Hello"      # str
is_valid = True        # bool

Basic Operations

Python supports standard arithmetic operations:

# Arithmetic operations
a = 10
b = 3

print(a + b)   # Addition (13)
print(a - b)   # Subtraction (7)
print(a * b)   # Multiplication (30)
print(a / b)   # Division (3.333...)
print(a // b)  # Floor division (3)
print(a % b)   # Modulus (1)
print(a ** b)  # Exponentiation (1000)
