Python Data Structures

Learn about the fundamental data structures in Python for organizing and storing data.

Lists

Lists are ordered, mutable collections that can hold items of different types.

# Creating and using lists
fruits = ["apple", "banana", "cherry"]
print(fruits[1])  # Access by index (banana)

# Modifying lists
fruits.append("orange")  # Add item
fruits[0] = "kiwi"      # Change item
fruits.remove("banana")  # Remove item

# List operations
numbers = [1, 2, 3] + [4, 5]  # Concatenation
print(len(numbers))            # Length (5)

Tuples

Tuples are ordered, immutable collections similar to lists but cannot be changed after creation.

# Creating and using tuples
coordinates = (10.0, 20.0)
print(coordinates[0])  # Access by index (10.0)

# Tuples are immutable
# coordinates[0] = 15.0  # This would cause an error

# Common uses
def get_dimensions():
    return (800, 600)  # Often used to return multiple values

width, height = get_dimensions()  # Tuple unpacking

Dictionaries

Dictionaries store key-value pairs and provide fast lookups by key.

# Creating and using dictionaries
person = {
    "name": "Alice",
    "age": 25,
    "is_student": True
}

# Accessing values
print(person["name"])  # Alice
print(person.get("age"))  # 25 (safer access)

# Modifying dictionaries
person["age"] = 26           # Update value
person["occupation"] = "Developer"  # Add new key-value
del person["is_student"]     # Remove key

Sets

Sets are unordered collections of unique elements, useful for membership tests and eliminating duplicates.

# Creating and using sets
colors = {"red", "green", "blue"}
colors.add("yellow")  # Add element
colors.remove("red")  # Remove element

# Set operations
a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)  # Union: {1, 2, 3, 4, 5}
print(a & b)  # Intersection: {3}
print(a - b)  # Difference: {1, 2}
