Python Modules & Packages

Learn how to organize and reuse your code with modules and packages.

Creating and Using Modules

Modules are Python files containing reusable code.

# File: calculator.py
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

# In another file:
import calculator

result = calculator.add(5, 3)
print(result)  # 8

# Alternative imports
from calculator import multiply
print(multiply(4, 6))  # 24

Standard Library Modules

Python comes with many useful built-in modules.

import math
print(math.sqrt(16))  # 4.0

import random
print(random.randint(1, 10))  # Random number between 1-10

from datetime import datetime
print(datetime.now())  # Current date and time

Creating Packages

Packages are collections of modules in directories.

# Directory structure:
# mypackage/
# ├── __init__.py
# ├── module1.py
# └── module2.py

# Using the package:
from mypackage import module1
from mypackage.module2 import some_function

Installing Third-Party Packages

Using pip to install packages from PyPI.

# In your terminal/command prompt:
# pip install package-name

# Common pip commands:
# pip install requests     # Install a package
# pip list                # List installed packages
# pip freeze > requirements.txt  # Save dependencies
# pip install -r requirements.txt # Install from file

Virtual Environments

Isolating project dependencies.

# Creating a virtual environment
# python -m venv myenv

# Activating it (Windows):
# myenv\Scripts\activate

# Activating it (macOS/Linux):
# source myenv/bin/activate

# Deactivating:
# deactivate
