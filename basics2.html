Java Basics

Variables and Data Types

int age = 25;
double price = 19.99;
char grade = 'A';
boolean isJavaFun = true;

Java is statically typed, meaning variables must be declared with a specific type.

Control Structures

// If-else statement
if (score >= 90) {
    grade = 'A';
} else if (score >= 80) {
    grade = 'B';
}

// For loop
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
